## Fuxi Full Bootstrap (Ports & Adapters)
1) Unzip at repo root.
2) Merge `patches/tsconfig.paths.additions.json` into `tsconfig.json`.
3) Ensure Tailwind deps: `npm i -D tailwindcss postcss autoprefixer`.
4) Clean & run: `rm -rf .next .turbo && npm run dev`.
Routes: `/` start, `/project/demo/scoring` grid.
