## **1. Initiate: Define or Import the Capability Map**
**Goal:** Establish a clear starting point for capability modeling.  
**User actions:**

**AI Enhancements:**

**Goal:** Evaluate the current maturity, importance, and strategic impact.

**AI Enhancements:**
- Predict likely maturity from textual descriptions.
